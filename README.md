# 🚦 Traffic Violations Insight System
**EDA, Data Cleaning & Interactive Streamlit Dashboard**

## Setup
1. `pip install -r requirements.txt`
2. Download dataset: https://data.montgomerycountymd.gov/api/views/4mse-ku6q/rows.csv → `raw_traffic.csv`
3. `python clean_data.py`
4. `streamlit run app.py`

## Features
- Cleans 1M+ rows (dedupes SeqID, fixes dates/booleans/geo)
- EDA: Top violations, accident hotspots, time trends
- Interactive dashboard: Filters, Plotly charts, Folium map
